# templates/ — copy map

Blank-slate starters. Copy into your repo, fill the `<angle-bracket>`
slots, delete nothing else:

| Template | Destination in your repo |
|---|---|
| `control-README.md` | `control/README.md` |
| `inbox.md` | `control/inbox.md` |
| `status.md` | `control/status.md` |
| `outbox.md` | `control/outbox.md` |
| `claims-README.md` | `control/claims/README.md` |
| `claim-file.md` | `control/claims/<branch-or-scope>.md` — one per claim |
| `session-card.md` | `.sessions/<date>-<slice>.md` — one per session |

Writer ownership travels with the files: once copied, the inbox
belongs to the manager and everything else to the lane/session that
the writer table in `control/README.md` says.
