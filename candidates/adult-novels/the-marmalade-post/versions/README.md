Alternate cuts/angles/lengths/languages/formats of the same title live
here, one subdirectory per version. The EN master
(`../en/the-marmalade-post.md`, **15,040 words** measured by `wc -w`)
stays the canonical text; a version NEVER edits the master. This title is
Book One of a planned series — versions must keep series-recurring names
and the Book-2 hook (the Patience Crumb box) intact across editions.

## Convention

- **One subdir per version**, named for the version's angle (e.g. `nl/`).
  Each holds the version's complete manuscript file(s) — finished prose
  only, no placeholders, no summaries, self-contained arc — or, for
  production specs, a spec doc — plus notes (`NOTES.md` for manuscript
  versions: what was changed and why).
- **Specs/notes carry honest `wc -w` counts** (command shown; measured,
  never estimated), what was added/cut vs the base manuscript, and the
  version's market position — including every ⚑ owner gate.
- A version must not contradict the title's vetted positioning (series
  tag, blurb claims, content note — `../DECISIONS.md`,
  `docs/publishing/vetting/the-marmalade-post.md`) — if it would, the
  vetting packet gets a new owner decision first.
- **No version publishes anything.** Publishing stays owner-gated
  (`docs/publishing/CHECKLIST.md` §7, via the title's vetting packet).
