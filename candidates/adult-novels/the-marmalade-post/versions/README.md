Alternate cuts/angles/lengths/languages of the same title live here, one
subdirectory per version. The EN master (`../en/the-marmalade-post.md`,
15,040 words measured) stays the canonical text; a version NEVER edits
the master. This title is Book One of a planned series — versions must
keep series-recurring names and the Book-2 hook (the Patience Crumb box)
intact across editions.

## Convention

- **One subdir per version**, named for the version's angle
  (e.g. `nl/`).
- Each subdir contains exactly:
  - the **complete manuscript** for that version (finished prose only —
    no placeholders, no summaries, self-contained arc);
  - **`NOTES.md`** — what was changed and why, an honest `wc -w` of the
    version file, market position, and the standing ⚑ note that
    publishing stays owner-gated via the title's vetting packet
    (`docs/publishing/vetting/the-marmalade-post.md`).
- A version must not contradict the title's vetted positioning (series
  tag, blurb claims, content note) — if it would, the vetting packet
  gets a new owner decision first.
- Word counts are measured (`wc -w`), never estimated.
