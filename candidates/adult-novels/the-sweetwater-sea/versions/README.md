> **Status:** `active` — created 2026-07-14 (night run, ORDER 011 item 2
> remainder: NL editions of the catalog's EN-only adult titles).
> Convention per the the-weigh-house / the-night-kiln precedent; this

Alternate cuts/angles/formats/languages of the title live here, one
subdirectory per version — the title carries one complete EN novella
(`../en/the-sweetwater-sea.md`, **15,243 words** measured by `wc -w`).
The EN master stays the canonical text; a version NEVER edits a master.

## Convention

- **One subdir per version**, named for the version's angle (e.g. `nl/`).
  Each holds the version's complete manuscript file(s) — finished prose
  only, no placeholders, no summaries, self-contained arc — or, for
  production specs, a spec doc — plus notes (`NOTES.md` for manuscript
  versions: what was changed and why).
- **Specs/notes carry honest `wc -w` counts** (command shown; measured,
  never estimated), what was added/cut vs the base manuscript, and the
  version's market position — including every ⚑ owner gate.
- A version must not contradict the title's vetted positioning
  (`../DECISIONS.md`, `docs/publishing/vetting/the-sweetwater-sea.md`) —
  if it would, the vetting packet gets a new owner decision first.
- **No version publishes anything.** Publishing stays owner-gated
  (`docs/publishing/CHECKLIST.md` §7, via the title's vetting packet).
