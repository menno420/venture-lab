# Ultramarine — versions/

> **Status:** `active` — created 2026-07-13 (night run, ORDER 008 "multiple
> versions of each: different angles, audiences, lengths")

Derived editions of *Ultramarine* (Delft, 1654 — base manuscript
[`../manuscript/ultramarine.md`](../manuscript/ultramarine.md), 27,865 words
measured). Convention follows the catalog's existing versions dirs
(`candidates/childrens-books/*/versions/`):

- **One subdirectory per version.** Each holds the version's complete
  manuscript file(s) plus a `NOTES.md`.
- **NOTES.md carries honest `wc -w` counts** (command shown), what was
  added/cut vs the base manuscript, and the version's market position —
  including every ⚑ owner gate.
- **The base manuscript is the source of truth** for story canon
  (continuity: [`../bible/`](../bible/)); a version adapts presentation
  (angle, audience, length, format), never silently changes canon.
- **No version publishes anything.** Publishing stays owner-gated via
  [`docs/publishing/vetting/ultramarine.md`](../../../../docs/publishing/vetting/ultramarine.md).
