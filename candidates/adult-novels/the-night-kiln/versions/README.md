Alternate cuts/angles/languages of the same title live here, one
subdirectory per version. The EN master (`../en/the-night-kiln.md`,
15,999 words measured) stays the canonical text; a version NEVER edits
the master. Book Two (`../en/the-morning-door.md`) is a separate book,
not a version, and does not live here.

## Convention

- **One subdir per version**, named for the version's angle (e.g. `nl/`).
- Each subdir contains exactly:
  - the **complete manuscript** for that version (finished prose only —
    no placeholders, no summaries, self-contained arc);
  - **`NOTES.md`** — what was changed and why, an honest `wc -w` of the
    version file, market position, and the standing ⚑ note that
    publishing stays owner-gated via the title's vetting packet
    (`docs/publishing/vetting/the-night-kiln.md`).
- A version must not contradict the title's vetted positioning — if it
  would, the vetting packet gets a new owner decision first.
- Word counts are measured (`wc -w`), never estimated.
