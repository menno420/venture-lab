# The Night Kiln — versions/

> **Status:** `active` — created 2026-07-13 (night run, ORDER 011 item 2,
> relaying ORDER 008 item 1: "multiple versions of each — different angles,
> audiences, lengths; versions are cheap once the research exists")

Alternate cuts/angles/formats of the series live here, one subdirectory per
version **per manuscript** — the title carries two complete EN novellas
(Book One `../en/the-night-kiln.md`, **15,999 words** measured; Book Two
*The Morning Door* `../en/the-morning-door.md`, **15,995 words** measured by
`wc -w`). The EN masters stay the canonical texts; a version NEVER edits a
master.

## Convention

- **One subdir per version per manuscript**, named for the version's angle
  (Book Two dirs carry a `-book-2` suffix). Each holds the version's complete
  manuscript file(s) — or, for production specs, a spec doc — plus notes.
- **Specs/notes carry honest `wc -w` counts** (command shown), what was
  added/cut vs the base manuscript, and the version's market position —
  including every ⚑ owner gate.
- A version must not contradict the title's vetted positioning
  (`../DECISIONS.md`, `docs/publishing/vetting/the-night-kiln.md`) — in
  particular, the ⚑ OWNER length-band question on Book Two (DECISIONS.md,
  2026-07-13) stays open: versions spec the manuscripts as written.
- **No version publishes anything.** Publishing stays owner-gated
  (`docs/publishing/CHECKLIST.md` §7).
