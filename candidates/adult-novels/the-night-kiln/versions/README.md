Alternate cuts/angles/formats/languages of the series live here, one
subdirectory per version **per manuscript** — the title carries two complete
EN novellas (Book One `../en/the-night-kiln.md`, **15,999 words** measured;
Book Two *The Morning Door* `../en/the-morning-door.md`, **15,995 words**
measured by `wc -w`). The EN masters stay the canonical texts; a version
NEVER edits a master.

## Convention

- **One subdir per version per manuscript**, named for the version's angle
  (e.g. `nl/`; Book Two dirs carry a `-book-2` suffix). Each holds the
  version's complete manuscript file(s) — finished prose only, no
  placeholders, no summaries, self-contained arc — or, for production
  specs, a spec doc — plus notes (`NOTES.md` for manuscript versions:
  what was changed and why).
- **Specs/notes carry honest `wc -w` counts** (command shown; measured,
  never estimated), what was added/cut vs the base manuscript, and the
  version's market position — including every ⚑ owner gate.
- A version must not contradict the title's vetted positioning
  (`../DECISIONS.md`, `docs/publishing/vetting/the-night-kiln.md`) — if
  it would, the vetting packet gets a new owner decision first. In
  particular, the ⚑ OWNER length-band question on Book Two (DECISIONS.md,
  2026-07-13) stays open: versions spec the manuscripts as written.
- **No version publishes anything.** Publishing stays owner-gated
  (`docs/publishing/CHECKLIST.md` §7, via the title's vetting packet).
