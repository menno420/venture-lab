Alternate cuts/angles/lengths of the same title live here, one
subdirectory per version. The EN master (`../en/the-weigh-house.md`,
36,434 words measured) stays the canonical text; a version NEVER edits
the master.

## Convention

- **One subdir per version**, named for the version's angle
  (e.g. `novella-cut/`).
- Each subdir contains exactly:
  - the **complete manuscript** for that version (finished prose only —
    no placeholders, no summaries, self-contained arc);
  - **`NOTES.md`** — what was changed and why (scene-level cut/change
    list), an honest `wc -w` of the version file, market position with
    price band per `docs/publishing/CHECKLIST.md` §4, and the standing
    ⚑ note that publishing stays owner-gated via the title's vetting
    packet (`docs/publishing/vetting/the-weigh-house.md`).
- A version must not contradict the title's vetted positioning
  (subtitle decision, blurb claims, content note) — if it would, the
  vetting packet gets a new owner decision first.
- Word counts are measured (`wc -w`), never estimated.
