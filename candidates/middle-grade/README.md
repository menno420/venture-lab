# Middle Grade

Original middle-grade candidates (ages 8–12, novel length), one slug directory per work.
Convention mirrors `candidates/adult-novels/`:

- `<slug>/DECISIONS.md` — creative decision log (decide-and-flag).
- `<slug>/CANON.md` — series canon sheet (rules, geography, cast, open hooks), updated in the same PR as each book lands.
- `<slug>/en/` — English manuscript. (Translations are follow-on scope.)

## Works
- `the-halfway-ferry/` — MG fantasy series opener (Book 1: The Fogline Crossing). A ferry with no name and no timetable docks at Little Slake's condemned jetty, and eleven-year-old Wren Tolliver does the sensible thing: she asks for a job.
