# Die Laternentür — Pappbilderbuch-Ausgabe

- **Title:** Die Laternentür (Pappbilderbuch-Fassung)
- **Language:** Deutsch (native parallel version — keine wörtliche Übersetzung)
- **Age band:** Pappbilderbuch, 0–3 Jahre
- **Format:** 12 Doppelseiten, eigenständiges Vorlesebuch für die Kleinsten — KEINE Zusammenfassung des Bilderbuchs
- **Word count:** honest `wc -w` per language in [`NOTES.md`](NOTES.md)
- **Pitch:** Juno findet eine winzige Tür, eine winzige Stadt und eine dunkle Laterne — und mit eins, zwei, drei kleinen Dingen macht sie sie wieder an.

> Fassung des Bilderbuchs für 4–8 Jahre (`../../the-lantern-door.de.md`). Gleiche Figuren (Juno, Docht), ein Kernmoment bewahrt: die kleine Juno entzündet die Große Laterne neu und macht die winzige Stadt wieder warm. Jede Doppelseite trägt eine Illustrationsnotiz in Kursiv; der Seat erzeugt keine Bilder.

---

[Spread 1]
*Illustration: Der moosige Fuß eines hohen Leuchtturms in der Dämmerung; eine buttergelbe Minitür; Juno kauert mit erhobenen Knöcheln.*
Ganz unten am großen Leuchtturm findet Juno eine winzige gelbe Tür. Klopf, klopf, klopf!

[Spread 2]
*Illustration: Die kleine Tür weit offen, warmes goldenes Licht strömt heraus; drinnen glüht eine Stadt so groß wie eine Teetasse.*
Die kleine Tür schwingt auf. Warmes goldenes Licht! Hallo, kleine Stadt.

[Spread 3]
*Illustration: Juno nun winzig in einer Kopfsteingasse, begrüßt von Docht mit rußiger Nase und langer Laternenstange.*
Juno tritt hindurch — und jetzt ist Juno auch klein! "Hallo! Ich bin Docht."

[Spread 4]
*Illustration: Der dunkle Turm über fröstelnden Stadtleuten in Schals; eine fingerhutkleine Katze an Junos Knöchel.*
"Die Große Laterne ist aus," sagt Docht. "Brrr. Die ganze Stadt ist dunkel und kalt."

[Spread 5]
*Illustration: Docht zählt an drei Fingern vor einer Kreidezeichnung: ein Funke, ein Faden, eine Windschleife.*
"Wir brauchen drei kleine Dinge: einen Funken, einen Faden, einen Hauch Wind. Findest du sie, Juno?"

[Spread 6]
*Illustration: Eine mondbeschienene Mooswiese; Juno formt ihre Hände um einen schläfrigen Glühkäfer.*
Auf der Mooswiese blinzelt ein schläfriger Glühkäfer. Ein Funke! Das ist eins.

[Spread 7]
*Illustration: Unter einer kleinen Steinbrücke spinnt eine freundliche Spinne Silber in Junos Finger.*
Unter der Brücke spinnt eine freundliche Spinne. Ein Silberfaden! Das ist zwei.

[Spread 8]
*Illustration: Ein windiger Hügel; Juno fängt einen Windstoß in einer geschwungenen Muschel, das Haar im Wind.*
Auf dem windigen Hügel — wusch! — fängt Juno Wind in einer Muschel. Das ist drei!

[Spread 9]
*Illustration: Die Wendeltreppe windet sich hinauf ins Dunkel; Juno klettert über kniehohe Stufen, Docht direkt hinter ihr.*
Jetzt den Turm hinauf. Hoch, hoch, hoch! Große Stufen. Mutige Juno. Hoch, hoch, hoch!

[Spread 10]
*Illustration: Oben flammt die Große Laterne auf — goldenes Licht überall.*
Faden... Funke... wuuusch! Die Große Laterne wacht auf. Gold! Warm! Hell!

[Spread 11]
*Illustration: Licht strömt über alle Dächer; winzige Stadtleute jubeln mit rosigen Wangen; Docht dreht fröhliche Kreise.*
Licht in jedem Fenster. Licht in jeder Gasse. Die kleine Stadt ist wieder warm. Hurra für Juno!

[Spread 12]
*Illustration: Juno wieder groß draußen in der Dämmerung, eine beerengroße Laterne glüht in ihrer Tasche; die kleine Tür klickt zu.*
Zurück durch die winzige Tür geht Juno. In ihrer Tasche: eine winzige Laterne, noch warm. Gute Nacht, kleine Stadt. Gute Nacht, Juno.

*Ende.*
