## Convention

Each title is a slug folder mirroring [`../childrens-books/`](../childrens-books/): a top-level `README.md` (index + word-count table) and `DECISIONS.md` (creative calls + rationale), then a language folder (`en/`) holding a per-language `README.md` (pitch, age band, chapter list, content note), the numbered chapter files (epigraph `00`, chapters `01`–`15`, epilogue `16`), and an `EXPANSION.md` (a one-page "if expanded to a full novel" note). Localization (NL/DE) is a per-title owner-approval follow-up, not included by default.

## Status

These are creative manuscripts, not shippable products: cover art, typesetting, and every publish/print/listing path remain owner-gated OWNER-ACTIONs. No images were generated; no revenue is claimed. See [INTAKE-lite.md](INTAKE-lite.md) for honest scope.
