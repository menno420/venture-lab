> **Status:** `active` — created 2026-07-13 (night run, ORDER 008 "multiple

Derived editions of *The Undertow* (YA dystopian SF, ages 13–17 — base
manuscript [`../en/`](../en/), files `00`–`16` in numeric order,
**27,781 words** measured: `wc -w` on `en/*.md` totals `28653`, minus
`EXPANSION.md` (`410`) and `README.md` (`462`), which are not story text).
Convention follows the catalog's existing versions dirs
(`candidates/adult-novels/{ultramarine,the-slow-word}/versions/`):

- **One subdirectory per version.** Each holds the version's complete
  manuscript file(s) — or, for production specs, a spec doc — plus notes.
- **Specs/notes carry honest `wc -w` counts** (command shown), what was
  added/cut vs the base manuscript, and the version's market position —
  including every ⚑ owner gate.
- **The base manuscript is the source of truth** for story canon
  (continuity: [`../DECISIONS.md`](../DECISIONS.md)); a version adapts
  presentation (angle, audience, length, format), never silently changes
  canon.
- **No version publishes anything.** Publishing stays owner-gated
  (`docs/publishing/CHECKLIST.md` §7).
