## Long description
```
The Agent-Workflow Template Pack is a small, honest scaffold for teams running Claude Code or other agent sessions. It gives you a starter agent constitution with fill slots, a session-log card template, a one-page discipline playbook, and three advisory Claude Code hooks — all plain Markdown and shell, dropped straight into your repo.

The point is a shared shape: how each session is logged, how work is claimed before it's built so parallel agents don't collide, and a quality-floor rule that says what "done" means. You customize the templates to your team and keep them in version control alongside your code.

This is an honest v0.x. The hooks are advisory and fail open — they print reminders and exit 0. Nothing here enforces anything or gates a commit. It is conventions and reminders you adopt, not a policy engine.
```
